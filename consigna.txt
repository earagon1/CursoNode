Utilizando express + Mongo DB Desarrollar los servicios para un ecommerce (sin carrito de compra) :

    Home
        Que muestra un listado de productos
    Pagina de detalle de producto
        Deberá retornar un json con los siguientes datos
            Nombre
            Precio
            Código
            Descripción
            Categoría
    Registro 
        Alta de usuario
    Login
        Validar el usuario enviando un json por POST

Se debe utilizar JWT para la generación del token
